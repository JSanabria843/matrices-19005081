1.3 Demuestre que (A + B) + C = A + (B + C):

![imagen3](https://github.com/user-attachments/assets/c7b5187e-4848-4d27-9457-a95f000348e0)

Calcular (A+B) +C 
A+B  =  [1+4    0+(-1) ]         =     [5     -1]
	      [-2+2    3+0    ]		   [0      3]
       
Se suma (A+B) +C
(A+B) +C =   [5   -1]        +      [3      2]        = [5+3      -1+2  ]
	           [0    3] 	            [-1     1]	        [0+(-1)    3+1]

            (A+B) +C = [8     1]
	               [ -1  4]	

Se calcula A + (B+C)

Sumamos:

B+C = [4     -1]      +     [3      2]   =      [4+3       -1+2]
      [2      0]	    [-1     1]	        [2+(-1)    0+1]
			B+C=  [7     1]
			      [1     1]

  Terminamos sumando A+(B+C)
  
A+ (B+C)   =   [1    0]     +    [7     1]      =    [1+7      0+1]
	       [-2  3]           [1      1] 	     [-2+1    3+1]

	     A+(B+C) = [8    1]
		       [-1   4]
